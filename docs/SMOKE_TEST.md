# SMOKE TEST (ручная проверка)

Ниже — базовый чеклист для ручной проверки после изменений.

1. **Запуск проекта**
   - Шаг: Запустить проект из Godot.
   - Ожидаемый результат: Проект стартует без ошибок, отображается стартовая сцена.

2. **Появление Login UI**
   - Шаг: Дождаться загрузки стартовой сцены.
   - Ожидаемый результат: Отображается экран логина.

3. **Прохождение UI-флоу**
   - Шаг: Ввести логин/пароль и нажать Login.
   - Ожидаемый результат: Переход к экрану выбора/создания персонажа.

4. **Загрузка персонажа / мира**
   - Шаг: Создать персонажа (если нужно), выбрать его и нажать Enter.
   - Ожидаемый результат: Загружается игровая сцена (мир).

5. **Базовые интеракции (если есть)**
   - Шаг: Проверить базовые взаимодействия (например, перемещение, выбор цели кликом).
   - Ожидаемый результат: Базовые интеракции работают без ошибок.
