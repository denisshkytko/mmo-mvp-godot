# Warrior spell balance pass (2026-03) / Баланс-проход по заклинаниям воина (2026-03)

## RU

Этот проход охватывает **всю линейку способностей воина** по общей системе балансировки: throughput, tempo, resource pressure и role identity.

Цель: укрепить идентичность воина как агрессивного frontliner-класса с сильным burst-окном через стойки, но без просадки полезности в длинном бою.

### Изменения по всем способностям

#### Core damage / pressure
- **Powerful Strike**: `110..145 -> 120..176`, cooldown `5.0 -> 4.5`, cost `5 -> 6`.
- **Armor Break**: DoT pct `60..120 -> 70..130`, cooldown `10 -> 9`, cast `1.0 -> 0.0`, cost `10 -> 11`.
- **Sweeping Strikes**: `100/100 -> 115/135`, cooldown `10/8 -> 9/7`, cost `15 -> 16`.
- **Dirty Strike**: `100% -> 120%` weapon scaling, cooldown `10 -> 9`.

#### Stances / utility / passives
- **Battle Readiness**: physical damage bonus `15% -> 18%`.
- **Polished Armor**: defense `15% -> 18%`, retaliate speed debuff `5% -> 6%`.
- **Boiling Blood**: clearer risk/reward curve (incoming damage penalty `30/25/20`, crit `10/15/20 -> 12/18/24`, crit damage `20/30/40 -> 24/34/44`).
- **Inspiration**: max HP bonus `20/40 -> 25/45`.
- **Defensive Reflexes**: passive reset cooldown `8/6 -> 7/5`.

### Ожидаемый эффект

- Базовые кнопки урона дают более стабильный темп и лучше масштабируются в поздних рангах.
- ДПС-стойки и defensive-опции сильнее различаются по роли, но без «обязательной» единственной стойки.
- В длительном бою warrior сохраняет pressure благодаря более практичным кд/кастам ключевых умений.

## EN

This pass covers the **entire warrior ability kit** with the shared balancing framework: throughput, tempo, resource pressure, and role identity.

Goal: reinforce warrior identity as an aggressive frontline class with clear burst windows through stance choices, without falling off in longer fights.
