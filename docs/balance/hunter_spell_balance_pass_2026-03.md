# Hunter spell balance pass (2026-03) / Баланс-проход по заклинаниям стрелка (2026-03)

## RU

Этот проход покрывает **всю линейку способностей стрелка** по той же методике, что применялась к магу/жрецу/храмовнику/шаману: throughput, tempo, resource pressure и role identity.

Цель: закрепить роль стрелка как устойчивого ranged-dps класса с понятным выбором аспектов (крит / скорость атаки / мобильность), без узкого «единственно верного» спама одной кнопки.

### Изменения по всем способностям

#### Core damage и ротация
- **Arcane Shot**: `5..80 -> 8..120`, cost `8 -> 9`.
- **Aimed Shot**: `150/200/250 -> 165/225/285`, cast `2.0/3.0/4.0 -> 1.8/2.6/3.4`, cost `16 -> 18`.
- **Poisoned Arrow**: DoT flat `20..140 -> 30..210`, cooldown `10 -> 9`, cost `12 -> 13`.
- **Suppressive Shot**: `60/70/80 -> 70/85/100`, cast `2.0 -> 1.8`, cost `16 -> 18`.
- **Panjagan**: `40/60/80 -> 50/75/100`, cast `3.0/4.0/5.0 -> 2.6/3.4/4.2`, cooldown `15 -> 14`, cost `16 -> 18`.
- **Lucky Shot**: cooldown `60 -> 45`, cast `2.0/3.0 -> 1.8/2.4`, mana return `20/30 -> 18/26`.

#### Aspects / stances
- **Cobra Aspect**: crit `5..30 -> 6..33`.
- **Mantis Aspect**: attack speed `10..40 -> 12..42`, speed multiplier `1.10..1.40 -> 1.12..1.42`.
- **Falcon Aspect**: move speed `30 -> 35`.

### Ожидаемый эффект

- Базовая ротация получает более ровный рост отдачи по рангам и меньше «провалов» на высоких рангах долгих кастов.
- Периодические/контрольные кнопки лучше окупаются по отношению к GCD, кд и стоимости.
- Аспекты дают более внятный стиль игры (burst/tempo/mobility), без потери sustain в длительном бою.

## EN

This pass covers the **entire hunter ability kit** with the same framework used for mage/priest/paladin/shaman: throughput, tempo, resource pressure, and role identity.

Goal: keep hunter as a consistent ranged-DPS class with clear aspect choices (crit / attack speed / mobility), without collapsing gameplay into a single dominant spam button.
